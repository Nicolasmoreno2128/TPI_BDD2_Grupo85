Explicación del Sistema
El siguiente sistema está desarrollado para gestionar de manera eficiente un taller mecánico en el que se realizan servicios y mantenimiento de automóviles. 
En el mismo se podrán registrar y administrar los turnos  para las revisiones, cancelarlos o modificarlos y almacenar toda la información relacionada con los vehículos y los propietarios.
Además facilita el control de los services, detallando qué tipo de service fue realizado, las fechas de inicio y final y quien fue el empleado que lo realizó, evitando así errores y problemas a futuro, asegurando la trazabilidad de los trabajos.
La base de datos almacenará información detallada sobre clientes, sus vehículos, los empleados del taller, los turnos agendados y el historial completo de cada servicio realizado, incluyendo los repuestos utilizados y la mano de obra aplicada.
El sistema cuenta con un inicio de sesión de usuario y contraseña desde el cual cada persona ingresa dependiendo su rol, disponiendo de diferentes funcionalidades según su perfil.

Funcionalidades principales
Gestión de clientes y vehículos: Permite registrar, modificar y consultar la información de los clientes y sus vehículos asociados (patente, marca, modelo, kilometraje).

Gestión de Empleados: Registrar los datos del personal y sus roles dentro del taller

Gestión de Turnos: Permite tomar turnos nuevos, modificar, cancelar o eliminar turnos existentes y consultar el estado (pendiente, completado).

Gestión de servicios: Permite almacenar el detalle de cada servicio realizado, asociando al empleado que lo realizó, los repuestos utilizados y la mano de obra aplicada

Gestión de inventario: Permite administrar el catálogo de repuestos y actualización del stock automáticamente.

Informes: Se van a realizar informes detallados de todas las entidades de la empresa permitiendo visualizar las estadísticas en un periodo determinado de tiempo y asi facilitar la toma de decisiones en la empresa. 

Este sistema fue diseñado mediante el sistema de base de datos relacionales que almacena toda la información necesaria para permitir estas funcionalidades.
